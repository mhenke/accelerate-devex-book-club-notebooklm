# AI Tools Used in This Project

This document provides a comprehensive list of the AI tools and services used to develop and maintain the Accelerate DevEx Book Club project.

## Content Generation and Research

### Google NotebookLM

- **Purpose:** Used to generate supplementary educational content, including podcasts and video scripts, based on the "Accelerate" book and other source materials.
- **Official Site:** [notebooklm.google.com](https://notebooklm.google.com)

### Claude (Anthropic)

- **Purpose:** Instrumental in developing the book club's structure, writing guides, creating discussion questions, and generating meeting templates.
- **Official Site:** [claude.ai](https://claude.ai)

## Code Development and Implementation

### Gemini

- **Purpose:** The Gemini family of models from Google played a crucial role in the project's development, assisting with code generation, refactoring, and providing guidance on best practices.
- **Official Site:** [gemini.google.com](https://gemini.google.com)

### Claude Code (Anthropic)

- **Purpose:** Used for hands-on code development, implementing the Jekyll-based website, and managing the repository's automation and tooling.
- **Official Site:** [claude.ai/code](https://claude.ai/code)

### GitHub Copilot

- **Purpose:** Provided AI-powered code completion and assistance within the IDE, helping to accelerate development and write documentation.
- **Official Site:** [github.com/features/copilot](https://github.com/features/copilot)

### Google Jules

- **Purpose:** An autonomous AI coding agent that integrates with GitHub workflows to fix bugs, update dependencies, migrate code, and add features. Uses Gemini 3 Pro for advanced coding reasoning and operates asynchronously in cloud VMs.
- **Official Site:** [jules.google](https://jules.google)

### Google Antigravity

- **Purpose:** An agentic development platform and AI-powered IDE that enables autonomous AI agents to plan, execute, and verify complex coding tasks. Built as a fork of VS Code with support for multiple AI models including Gemini 3 Pro and Claude Sonnet 4.5.
- **Official Site:** [antigravity.google](https://antigravity.google)



## Visuals and Presentations

### Gamma

- **Purpose:** Used to create presentations, slide decks, and other visual storytelling content for the book club meetings.
- **Official Site:** [gamma.app](https://gamma.app)

### Napkin AI

- **Purpose:** Assisted in the creation of visual content, diagrams, and infographics to explain complex concepts from the book.
- **Official Site:** [napkin.ai](https://www.napkin.ai/)
