# Activities & Exercises

Supporting activities for the Accelerate DevEx Book Club to help teams apply concepts between meetings.

## Core Activities

- **[Discussion Practices](discussion-practices.md)** - Guidelines for creating psychological safety and productive conversations
- **[Between Meetings Activities](between-meetings.md)** - Structured exercises to try between book club sessions
- **[Facilitation Guide](facilitation-guide.md)** - Best practices for leading book club discussions

## Purpose

These activities support the book's core message: you can't "install" culture change - you have to **act your way into a better culture** through deliberate practice and experimentation.

## How to Use

1. **Before meetings:** Review discussion practices to set expectations
2. **Between meetings:** Try small experiments from the activities list
3. **During meetings:** Use facilitation guide for smooth discussions
4. **Ongoing:** Implement WIP limits and other lean practices
